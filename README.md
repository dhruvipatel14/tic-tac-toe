# Tic Tac Toe


## Live project link : [tic-tac-toe](https://tic-tac-toe-o-x.netlify.app/)

The project demonstrates the implementation of a two player tic-tac-toe game.

### Tech stack:
- React
- HTML
- CSS

### Features:
- Current player tracking
- Game logging
- Winning combinations
- Rematch
